1. Acessar o pfSense
2. Criar o arquivo /usr/local/www/css/modern-gui.css
    -  Acessar Diagnotics>Command Prompt e executar o comando touch /usr/local/www/css/modern-ui.css
3. Adicionar o conteudo do arquivo css no /usr/local/www/css/modern-ui.css
    - Acesse Diagnotics>Edit File e carregue o arquivo /usr/local/www/css/modern-ui.css, então preencha e salve
4. Alterar o tema para modern-gui em System>General Setup>webConfigurator>Theme

recomendado para 1920x1024 +
